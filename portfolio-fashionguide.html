The Fashion Guide: E-closet Database built with the Flask Framework

The Fashion Guide is a webapp that stores the user's outfits and becomes a future reference to access recomended outfits. In other words, The Fashion Guide is an outfit database that allows a user to store an outfit, rate it, and comment on how warm (or not warm) the outfit is. To use fashion guide, all one needs is a picture of their outfit. They will simply upload that picture to fashion guide, rate it, and indicate how warm it is. On the longterm, this outfit database will accrue to become a daily reference of what the user wants to wear based on their rating and/or on the necessity of warmth.
